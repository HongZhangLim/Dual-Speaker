# 🔊 Dual-Speaker Magisk Module 🔊 [![Download ZIP](https://img.shields.io/badge/Download-Module.zip-2ea44f?style=for-the-badge&logo=android&logoColor=white)](https://github.com/TheWhiteLHZ/Dual-Speaker/files/5030540/1.Note7.zip)

**Enable stereo audio by utilizing your device's earpiece as a secondary speaker.**
**通过将设备的听筒作为副扬声器使用，实现立体声音效。**

This module modifies your system's audio configuration to allow the top earpiece and the main bottom speaker to play audio simultaneously, creating a dual-speaker (stereo) effect.

本模块通过修改系统音频配置，使顶部听筒与底部主扬声器同时发声，从而营造出双扬声器（立体声）效果。

## 🛠 Installation / 安装步骤

1.  Open the **Magisk App**.
2.  Go to the **Modules** tab.
3.  Select **Install from storage** and pick the `.zip` file.
4.  **Reboot** your device.


**安装说明：**
1.  打开 **面具 App**。
2.  进入 **模块** 选项卡。
3.  选择 **从本地安装**，然后选中下载的 `.zip` 压缩包。
4.  **重启** 设备。
